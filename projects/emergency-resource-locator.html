Emergency Resource Locator

🐊 UF SwampHacks XI

An interactive map that helps people quickly find nearby emergency services and critical resources during active crises — hospitals, shelters, pharmacies, and more — all in one place.

The Problem

During emergencies, people scramble across multiple apps and websites to locate hospitals, shelters, fire stations, and pharmacies. Every second counts, and fragmented information costs lives. Emergency Resource Locator consolidates these into a single, fast-loading map view.

How It Works

The app uses the browser's Geolocation API to detect the user's position, then renders nearby emergency resources on an interactive Leaflet.js map. Resources are color-coded by type, and clicking a marker shows contact info and directions. The interface is intentionally minimal — designed to work under stress with no learning curve.

Built in 24 hours at SwampHacks XI, the project prioritized speed of use over feature breadth. The React Native web target and Vite build system kept iteration cycles fast throughout the hackathon.

Tech Stack

React Native JavaScript Leaflet.js Vite Geolocation API OpenStreetMap
