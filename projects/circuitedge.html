Continuous Project

CircuitEdge

Active Development

An Arduino micro-learning SaaS platform with OAuth authentication, Stripe payments, and a Supabase backend — deployed on Cloudflare Pages for global edge performance.

What It Does

CircuitEdge delivers bite-sized Arduino programming lessons directly in the browser. Users progress through structured modules, write code in an embedded editor, and get instant feedback — no IDE setup required.

The platform supports both free and paid tiers. Stripe handles subscriptions and one-time purchases, while Supabase manages user data, progress tracking, and real-time updates. Google and GitHub OAuth let users sign in without a password.

Infrastructure

Cloudflare Pages — static frontend with edge functions for low-latency API responses worldwide.

Supabase — PostgreSQL database with row-level security, auth integration, and real-time subscriptions.

Stripe — subscription billing, webhooks for lifecycle events, and a customer portal for self-service management.

Tech Stack

Cloudflare Pages Supabase Stripe OAuth (Google + GitHub) Arduino PostgreSQL
